## Changelog Summary

### Core

- local-first workspace and case storage with deterministic replay
- event-log-backed mutation flow for nodes, edges, tasks, decisions, events, and evidence
- storage validation, event verification, and cache rebuild commands

### Patch / Sync

- `GraphPatch` validation, review, and apply flow
- built-in markdown sync as the required reference integration
- reverse sync from markdown into patch artifacts
- black-box E2E coverage for the documented Quickstart and Manual Acceptance core/sync flows

### Analysis

- frontier and blocker derivation
- impact and critical-path analysis
- slack, bottlenecks, unblock-set, cycles, components, bridges, cutpoints, and fragility surfaces

### Worker / Extension Surfaces

- importer, sink, and worker plugin protocols
- built-in shell, local-LLM, and code-agent worker bridges

### Read-Only Graph Inspection

- `cg case view` remains the public read-only working surface
- actionable items render as `!`
- completed items render as `✓`
- waiting items render as `→`
- blocked items render as `✗`
- repeated DAG nodes render as `= ... (shared)` to avoid duplicating full subtrees

## Known Non-Blockers

- broader TUI / graph-view work is still exploratory and unfrozen
- external sinks remain optional integrations, not release blockers
- raw topology / Betti surfaces remain experimental only
